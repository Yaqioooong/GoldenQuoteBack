# GOLDEN QUOTES
> 发现、收藏并分享最珍贵的文字瞬间

## 功能设计
- 书籍及金句展示功能
- 书籍及金句管理功能
- 金句收藏功能(目前接口已实现，页面尚未实现)
- 注册、登录功能(SaToken)
- 接口角色鉴权(SaToken)
## 技术选型
- SpringBoot
- MySQL
- MyBatis Plus
- SaToken
- Redis
- RESTful API
- MAVEN
- Next.js
